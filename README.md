# sibo-autoinmune-anti-CdtB-y-anti-vinculina
Las investigaciones del Dr. Mark Pimentel (Cedars-Sinai) revelan que gran parte del SIBO es de origen autoinmune. Una intoxicación alimentaria previa genera anticuerpos (anti-CdtB y anti-vinculina) que atacan los nervios del intestino. Esto daña el "complejo migratorio motor" (la limpieza intestinal), causando retención bacteriana
